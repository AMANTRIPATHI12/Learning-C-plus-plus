pass by value means when you give a variable as parameter to a function then it gets copied and put in place of the parameter which the function actully has.
For example in the main function you define two variables "a" , "b" with there values and then pass them in the add function to get their sum then the values of those "a" and "b" gets copied and the parameters "num1" and "num2" get those values and because of this if you change the values in the add function then it will not affect the values of variables inside main function.

Pass by reference means you pass the reference pointer of the orignal variable means you don't give the value you pass the current address of the value in the memory and because of it the add function can directly change the orignal value of orignal variable.

function overloading:- C++ allows function with same name but different parameter and return type. This is known as function overloading. The compiler itself decides which function needs to run based on the values provided as parameter.
example :- 
            int max(int a, int b) {
                return (a > b) ? a : b;
            }

            double max(double a, double b) {
                return (a > b) ? a : b;
            }

            int main() {
                cout << max(3, 7) << endl;      // calls the int version
                cout << max(2.5, 1.8) << endl;  // calls the double version
            }

Scope of variables
    variables declared inside a function are local and they can only be accessed inside that perticular block of code.
    variables declared outside a block usually at the top before main function or after iostream or before it is considered a global variable. They can be accessed anywhere in the code. But global variables might lead to messy code so use it effectivly.

Why use function :- Suppose you want to print the area of a circle for 100 different radius value then you need to write the same code again and again 100 times. But with a function you can write the code once and you can call it 100 times. and also Suppose you want to change something in the code of area of circle then you can change it once and it will reflect for all 100 radius. but if you don't use the function then you have to change for all 100 radius which is not efficient.
